# Restaurante Bottega

Aplicación web desarrollada en JavaScript para gestionar pedidos de un restaurante según el horario del día.

## Funcionalidades

- Menú dinámico según la hora
- Desayuno, almuerzo y cena
- Selección de platos
- Personalización de platos principales
- Extras adicionales
- Facturación automática
- Validación de horario y cancelación
- Recargo para menú de cena

## Tecnologías utilizadas

- HTML5
- JavaScript
- Moment.js
